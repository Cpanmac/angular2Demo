# angular2Demo
Angular 2 demo based on webpack.

## 使用方法
1. npm install
2. webpack --progress --colors
3. npm install webpack-dev-server -g --verbose
4. webpack-dev-server --content-base dist/ 或者 npm start
5. 访问localhost:8080
